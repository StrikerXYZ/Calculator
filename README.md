# No-CRT Calculator (Windows)

A calculator built directly on Win32 + Vulkan without the C runtime.

## Highlights

- No CRT
- Win32 platform layer
- Vulkan renderer
- Responsive anchor/grid layout
- Simulation separated from presentation
- Custom memory utilities
- String library
- Math library

## Run (Windows)

- Launch Calculator.sln
- Set Configutation to Release x64
- Run
